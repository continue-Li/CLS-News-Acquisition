# 财联社电报实时监听

每分钟自动拉取财联社电报，控制台实时输出当天最新快讯。

## 快速开始

```bash
# 编译 + 运行
mvn compile exec:java

# 或者双击
run.bat
```

按 `Ctrl+C` 停止。

## 工作原理

```
┌─ 每 60 秒一轮 ─────────────────────────┐
│                                         │
│  ① last_time = 当前时间 → 拉最新消息    │
│     └─ 有 → 输出，更新翻页锚点          │
│                                         │
│  ② 没新的 → last_time = 最早已读时间    │
│     └─ 往回翻更早的当天消息             │
│                                         │
│  ③ 两个方向都没了 → "暂无新的更新"      │
│                                         │
└─────────────────────────────────────────┘
```

- 优先展示最新发布，无新内容时自动回溯更早的消息
- 当天全部拉完后才提示"暂无新的更新"
- 跨天自动重置
